# Stripping Ratio Calculator
**Built by Carly Edward Quaicoe — UMaT Mining Engineering Year 1**

## What it does
Calculates key stripping ratios for open pit mining operations:
- Basic Stripping Ratio (BSR)
- Maximum Allowable Stripping Ratio (MASR)
- Automatic economic viability decision

## How to use
1. Open the .ipynb file in Google Colab
2. Run all cells
3. Enter your mine data when prompted
4. Results saved automatically to a text file

- Python 3
- Google Colab
